Unity 中的定时器与延迟函数踩坑记录

发布时间：2026-06-14 | 分类：游戏引擎

在 Unity 中实现定时计数时，经常会使用到 Invoke 和 InvokeRepeating。今天遇到了一个关于字符拼接的经典报错：

print(i++ + 's'); // 错误原因：单引号 's' 是 char 类型，相加时会变成 ASCII 码数学运算！

正确解法：应该使用双引号 print(i++ + "s"); 将其作为字符串拼接。同时，如果想要循环执行，应该使用 InvokeRepeating 而不是普通的 Invoke。
